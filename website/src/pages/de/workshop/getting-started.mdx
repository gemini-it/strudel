## Qu'est-ce que Strudel

Avec Strudel, vous pouvez écrire des morceaux de musique dynamiques en temps réel.
C'est une version développée en JavaScript de [Tidal Cycles](https://tidalcycles.org/) et a été initiée en 2022 par Alex McLean et Felix Roos.
Tidal Cycles, également connu sous le nom de Tidal, est un langage informatique pour les motifs algorithmiques.
Bien qu'il soit principalement utilisé pour la création musicale, il peut être utilisé pour tout type d'activité où les motifs jouent un rôle.

Vous n'avez pas besoin d'expérience en JavaScript ou en Tidal Cycles pour faire de la musique avec Strudel.
Cet atelier interactif vous guidera de manière ludique à travers les bases de Strudel.
Le meilleur endroit pour faire de la musique avec Strudel est le [Strudel REPL](https://strudel.cc/).

## Que pouvez-vous faire avec Strudel ?

- Live Coding de musique : Faire de la musique en temps réel avec du code
- Composition algorithmique : Écrire de la musique en utilisant le langage unique de Tidal pour les motifs
- Enseigner : Strudel est bien adapté aux enseignants, car aucune installation n'est nécessaire et le langage ne requiert aucune connaissance théorique préalable.
- Combiner avec d'autres applications musicales : Via MIDI ou OSC, Strudel peut être combiné avec n'importe quelle configuration en tant que séquenceur flexible.

## Exemple

Voici un exemple de ce à quoi Strudel peut ressembler :

<MiniRepl
  client:idle
  tune={`stack(
  // drums
  s("bd,[~ <sd!3 sd(3,4,2)>],hh*8")
  .speed(perlin.range(.8,.9)), // random sample speed variation
  // bassline
  "<a1 b1\*2 a1(3,8) e2>" 
  .off(1/8,x=>x.add(12).degradeBy(.5)) // random octave jumps
  .add(perlin.range(0,.5)) // random pitch variation
  .superimpose(add(.05)) // add second, slightly detuned voice
  .note() // wrap in "note"
  .decay(.15).sustain(0) // make each note of equal length
  .s('sawtooth') // waveform
  .gain(.4) // turn down
  .cutoff(sine.slow(7).range(300,5000)), // automate cutoff
  // chords
  "<Am7!3 <Em7 E7b13 Em7 Ebm7b5>>".voicings('lefthand') 
  .superimpose(x=>x.add(.04)) // add second, slightly detuned voice
  .add(perlin.range(0,.5)) // random pitch variation
  .note() // wrap in "note"
  .s('sawtooth') // waveform
  .gain(.16) // turn down
  .cutoff(500) // fixed cutoff
  .attack(1) // slowly fade in
)
.slow(3/2)`}
/>
