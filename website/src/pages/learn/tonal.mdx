# Fonctions tonales

Ces fonctions utilisent [tonaljs](https://github.com/tonaljs/tonal) pour fournir des aides aux opérations musicales.

### voicing()

<JsDoc client:idle name="voicing" h={0} />

Voici un exemple de comment vous pouvez jouer des accords et une ligne de basse :

<MiniRepl
  client:idle
  tune={`chord("<C^7 A7b13 Dm7 G7>*2")
  .dict('ireal').layer(
  x=>x.struct("[~ x]*2").voicing()
  ,
  x=>n("0*4").set(x).mode("root:g2").voicing()
  .s('sawtooth').cutoff("800:4:2")
)`}
/>

### transpose(semitones)

Transpose toutes les notes du nombre de demi-tons donné :

<MiniRepl client:only="react" tune={`"[c2 c3]*4".transpose("<0 -2 5 3>").note()`} />

Cette méthode devient vraiment passionnante quand nous l'utilisons avec un pattern comme ci-dessus.

Au lieu de nombres, la notation d'intervalle scientifique peut également être utilisée :

<MiniRepl client:only="react" tune={`"[c2 c3]*4".transpose("<1P -2M 4P 3m>").note()`} />

### scaleTranspose(steps)

Transpose les notes à l'intérieur de la gamme du nombre de pas :

<MiniRepl
  client:idle
  tune={`"[-8 [2,4,6]]*2"
.scale('C4 bebop major')
.scaleTranspose("<0 -1 -2 -3 -4 -5 -6 -4>*2")
.note()`}
/>

### rootNotes(octave = 2)

Transforme les symboles d'accords en notes fondamentales d'accords dans l'octave donnée.

<MiniRepl client:only="react" tune={`"<C^7 A7b13 Dm7 G7>*2".rootNotes(3).note()`} />

Avec layer, struct et voicings, cela peut être utilisé pour créer une piste d'accompagnement de base :

<MiniRepl
  client:idle
  tune={`"<C^7 A7b13 Dm7 G7>*2".layer(
  x => x.voicings('lefthand').struct("[~ x]*2").note(),
  x => x.rootNotes(2).note().s('sawtooth').cutoff(800)
)`}
/>
