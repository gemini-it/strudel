# Signaux continus

Les signaux sont des motifs avec des valeurs continues, ce qui signifie qu'ils ont théoriquement un nombre infini d'étapes.
Ils peuvent fournir des flux de nombres qui peuvent être échantillonnés à des points discrets dans le temps.
