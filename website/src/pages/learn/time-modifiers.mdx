# Modificateurs de temps

Les fonctions suivantes modifient d'une manière ou d'une autre la structure temporelle d'un motif.
Certaines d'entre elles ont des opérateurs équivalents dans la mini-notation :

| fonction               | mini         |
| ---------------------- | ------------ |
| `"x".slow(2)`          | `"x/2"`      |
| `"x".fast(2)`          | `"x*2"`      |
| `"x".euclid(3,8)`      | `"x(3,8)"`   |
| `"x".euclidRot(3,8,1)` | `"x(3,8,1)"` |
