# Récapitulatif de l'Atelier

Cette page est juste une liste de toutes les fonctions couvertes dans l'atelier !

## Mini-notation

| Concept           | Syntaxe  | Exemple                                                               |
| ----------------- | -------- | --------------------------------------------------------------------- |
| Séquence          | espace   | <MiniRepl client:visible tune={`sound("bd bd sd hh bd cp sd hh")`} /> |
| Numéro d'échantillon | :x    | <MiniRepl client:visible tune={`sound("hh:0 hh:1 hh:2 hh:3")`} />     |
| Silences          | ~        | <MiniRepl client:visible tune={`sound("metal ~ jazz jazz:1")`} />     |
| Sous-Séquences    | \[\]     | <MiniRepl client:visible tune={`sound("bd wind [metal jazz] hh")`} /> |
| Sous-Sous-Séquences | \[\[\]\] | <MiniRepl client:visible tune={`sound("bd [metal [jazz sd]]")`} />  |
| Accélérer         | \*       | <MiniRepl client:visible tune={`sound("bd sd*2 cp*3")`} />            |
| Parallèle         | ,        | <MiniRepl client:visible tune={`sound("bd*2, hh*2 [hh oh]")`} />      |
| Ralentir          | \/       | <MiniRepl client:visible tune={`note("[c a f e]/2")`} />              |
| Alterner          | \<\>     | <MiniRepl client:visible tune={`note("c <e g>")`} />                  |
| Allonger          | @        | <MiniRepl client:visible tune={`note("c@3 e")`} />                    |
| Répliquer         | !        | <MiniRepl client:visible tune={`note("c!3 e")`} />                    |

## Sons

| Nom   | Description                       | Exemple                                                                 |
| ----- | --------------------------------- | ----------------------------------------------------------------------- |
| sound | joue le son du nom donné          | <MiniRepl client:visible tune={`sound("bd sd")`} />                     |
| bank  | sélectionne la banque de sons     | <MiniRepl client:visible tune={`sound("bd sd").bank("RolandTR909")`} /> |
| n     | sélectionne le numéro d'échantillon | <MiniRepl client:visible tune={`n("0 1 4 2").sound("jazz")`} />       |

## Notes

| Nom       | Description                          | Exemple                                                                           |
| --------- | ------------------------------------ | --------------------------------------------------------------------------------- |
| note      | définir la hauteur en nombre ou lettre | <MiniRepl client:visible tune={`note("b g e c").sound("piano")`} />             |
| n + scale | définir la note dans la gamme        | <MiniRepl client:visible tune={`n("6 4 2 0").scale("C:minor").sound("piano")`} /> |
| $:        | jouer des patterns en parallèle      | <MiniRepl client:visible tune={'$: s("bd sd")\n$: note("c eb g")'} />             |

## Effets audio

| nom   | exemple                                                                                 |
| ----- | --------------------------------------------------------------------------------------- |
| lpf   | <MiniRepl client:visible tune={`note("c2 c3 c2 c3").s("sawtooth").lpf("400 2000")`} />  |
| vowel | <MiniRepl client:visible tune={`note("c3 eb3 g3").s("sawtooth").vowel("<a e i o>")`} /> |
| gain  | <MiniRepl client:visible tune={`s("hh*16").gain("[.25 1]*4")`} />                       |
| delay | <MiniRepl client:visible tune={`s("bd rim bd cp").delay(.5)`} />                        |
| room  | <MiniRepl client:visible tune={`s("bd rim bd cp").room(.5)`} />                         |
| pan   | <MiniRepl client:visible tune={`s("bd rim bd cp").pan("0 1")`} />                       |
| speed | <MiniRepl client:visible tune={`s("bd rim bd cp").speed("<1 2 -1 -2>")`} />             |
| range | <MiniRepl client:visible tune={`s("hh*32").lpf(saw.range(200,4000))`} />                |

## Effets de motif

| nom    | description                         | exemple                                                                             |
| ------ | ----------------------------------- | ----------------------------------------------------------------------------------- |
| setcpm | définit le tempo en cycles par minute | <MiniRepl client:visible tune={`setcpm(45); sound("bd sd [~ bd] sd")`} />         |
| fast   | accélérer                           | <MiniRepl client:visible tune={`sound("bd sd [~ bd] sd").fast(2)`} />               |
| slow   | ralentir                            | <MiniRepl client:visible tune={`sound("bd sd [~ bd] sd").slow(2)`} />               |
| rev    | inverser                            | <MiniRepl client:visible tune={`n("0 2 4 6").scale("C:minor").rev()`} />            |
| jux    | diviser gauche/droite, modifier droite | <MiniRepl client:visible tune={`n("0 2 4 6").scale("C:minor").jux(rev)`} />      |
| add    | ajouter nombres / notes             | <MiniRepl client:visible tune={`n("0 2 4 6".add("<0 1 2 1>")).scale("C:minor")`} /> |
| ply    | accélérer chaque événement n fois   | <MiniRepl client:visible tune={`s("bd sd").ply("<1 2 3>")`} />                      |
| off    | copier, décaler le temps & modifier | <MiniRepl client:visible tune={`s("bd sd, hh*4").off(1/8, x=>x.speed(2))`} />       |
