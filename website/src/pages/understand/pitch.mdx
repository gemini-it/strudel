# Comprendre la Hauteur

Apprenons comment fonctionne la hauteur ! Le curseur ci-dessous contrôle la <span style="color:#3b82f6;">fréquence</span> d'un oscillateur, produisant une hauteur :

{/* <PitchSlider client:load showFrequencySlider plot /> */}

<PitchSlider client:load showFrequencySlider min={20} max={20000} />

- Faites glisser le curseur pour entendre une hauteur
- Déplacez le curseur pour changer la hauteur
- Observez comment le nombre de Hz change
- <span className="text-red-300">Attention</span> : Les fréquences plus élevées pourraient être dérangeantes pour les enfants ou les animaux !

Le nombre de Hz est la fréquence de la hauteur que vous entendez.
Plus la fréquence est élevée, plus la hauteur est haute et vice versa.
Une hauteur se produit chaque fois que quelque chose vibre / oscille à une fréquence, dans ce cas c'est votre haut-parleur.
L'unité **Hz** décrit combien de fois cette oscillation se produit par seconde.
Nos yeux sont trop lents pour voir réellement l'oscillation sur le haut-parleur, mais nous pouvons <a href="https://www.youtube.com/watch?v=CDMBWw7OuJQ" target="_blank">la voir au ralenti</a>.

<Box>

La plage auditive d'un nouveau-né est censée être entre 20Hz et 20000Hz.
La limite supérieure diminue avec l'âge. Quelle est votre limite supérieure ?

</Box>

Dans Strudel, nous pouvons jouer les fréquences directement avec le contrôle `freq` :

<MiniRepl client:visible tune={`freq("<200 [300,500] 400 [500,<600 670 712 670>]>*8")`} />

## Fréquence vs Perception de la Hauteur

Peut-être avez-vous déjà remarqué que le <span style="color:#3b82f6;">curseur de fréquence</span> est "déséquilibré",
ce qui signifie que la hauteur change plus dans la région gauche et moins dans la région droite.<br/>
Pour rendre cela plus évident, ajoutons un <span style="color:#eab308">curseur de hauteur</span>
qui contrôle la fréquence sur une échelle différente :

<PitchSlider animatable plot showFrequencySlider showPitchSlider client:load />

Essayez les boutons ci-dessus pour balayer la plage de fréquence de 2 façons différentes :

- Balayage de Fréquence : <span style="color:#3b82f6;">la fréquence augmente de façon linéaire</span> , <span style="color:#eab308">la hauteur augmente de façon logarithmique</span>
- Balayage de Hauteur : <span style="color:#3b82f6;">la fréquence augmente de façon exponentielle</span> , <span style="color:#eab308">la hauteur augmente de façon linéaire</span>

<Box>

N'ayez pas peur de ces termes mathématiques :

- "logarithmique" est juste une façon élégante de dire "ça commence vite et ralentit"
- "exponentiel" est juste une façon élégante de dire "ça commence lentement et accélère"

</Box>

La plupart du temps, nous voulons contrôler la hauteur d'une manière qui correspond à notre perception,
ce que fait le <span style="color:#eab308">curseur de hauteur</span>.

## De Hz aux Demi-tons

Parce que Hz ne correspond pas à notre perception, essayons de trouver une unité pour la hauteur qui correspond.
Pour approcher cette unité de hauteur, regardons comment la fréquence se comporte lorsqu'elle est doublée :

<PitchSlider client:load showPitchSlider showFrequencySlider pitchStep={1 / 7} />

- Utilisez le curseur de hauteur maintenant échelonné ci-dessus
- Pouvez-vous entendre comment ces hauteurs semblent liées les unes aux autres ?

<Box>

En termes musicaux, une hauteur avec le double de la fréquence d'une autre est une `octave` plus haute.

</Box>

Parce que les octaves sont assez éloignées, les octaves sont généralement divisées en 12 parties plus petites :

<PitchSlider client:load showPitchSlider showFrequencySlider pitchStep={1 / 12} min={440} max={880} initial={440} />

Ce pas est également appelé un demi-ton, qui est la division la plus courante de la musique tonale.
Par exemple, les touches d'un clavier de piano sont également divisées en demi-tons.

Dans Strudel, nous pourrions faire cela avec `freq` comme ceci :

<MiniRepl
  client:visible
  tune={`freq(
  "0 4 7 12"
  .fmap(n => 440 * 2**(n/12))
)`}
/>

Bien sûr, cela peut être écrit plus court avec note, comme nous le verrons ci-dessous.

## Des Demi-tons aux Numéros MIDI

Maintenant nous savons quelle est la distance d'un demi-ton.
Ci-dessus, nous avons utilisé une fréquence de base arbitraire de 440Hz, ce qui signifie que l'exposant 0 est égal à 440Hz.
Typiquement, 440Hz est standardisé au numéro 69, ce qui conduit à ce calcul :

<PitchSlider
  client:load
  showPitchSlider
  showFrequencySlider
  baseFrequency={440}
  zeroOffset={69}
  pitchStep={1 / 12 / 7}
  min={440 / 8}
  max={7040}
  initial={440}
/>

Le nombre jaune est maintenant un numéro MIDI, couvrant plus que toute la plage auditive humaine avec des numéros de 0 à 127.
Dans Strudel, nous pouvons utiliser des numéros MIDI à l'intérieur de `note` :

<MiniRepl client:visible tune={`note("69 73 76 81")`} />

## Des Numéros MIDI aux Notes

Dans la théorie musicale occidentale, les notes sont utilisées au lieu des numéros.
Pour chaque numéro midi, il y a au moins une étiquette de note :

<PitchSlider
  client:load
  showPitchSlider
  showFrequencySlider
  baseFrequency={440}
  zeroOffset={69}
  pitchStep={1 / 48}
  min={440 / 8}
  max={880}
  initial={440}
  claviature
/>

Une étiquette de note complète se compose d'une lettre (A-G), 0 ou plusieurs altérations (b | #) et un numéro d'octave.
Ce système est également connu sous le nom de [Notation Scientifique de la Hauteur](https://en.wikipedia.org/wiki/Scientific_pitch_notation).
Dans Strudel, ces étiquettes de note peuvent également être utilisées à l'intérieur de `note` comme alternative aux numéros midi :

<MiniRepl client:visible tune={`note("A4 C#5 E5 A5").piano()`} />

</Box>
